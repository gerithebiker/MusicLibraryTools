MusicLibraryTools Installer by Geri

To install MusicLibraryTools, run the following command in PowerShell:

Invoke-WebRequest -Uri "https://raw.githubusercontent.com/gerithebiker/MusicLibraryTools/main/Install-MusicLibraryTools.ps1" -UseBasicParsing | Invoke-Expression

For more information, visit the MusicLibraryTools repository.
